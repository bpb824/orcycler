forestPlot

Usage

forestplot(model, xlab = "Odds Ratio", ylab = "Independent Variable", title, outputDir, rownames = NULL, xRng = NULL)

Arguments

model
Cumulative logistic regression model object
xlab
X-axis label, default = "Odds Ratio"
ylab
Y-Axis label, default = "Independent Variable"
title
Plot title
outputDir
Where to plot the .png output
rownames
Optional row names parameter
xRng
X-axis range

Value

None

Description

Create a forest plot to visualize cumulative logistic regression model coefficients
