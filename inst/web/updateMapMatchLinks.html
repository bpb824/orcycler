Update links and trips files from map matching.

Usage

updateMapMatchLinks(matchLinksPath, matchTripsPath)

Arguments

matchLinksPath
Relative or absolute path to existing matched link set.
matchTripsPath
Relative or absolute path to existing matched trip set.

Value

None

Description

This function updates map-matched links and trips files with results from python scripts stored in local PostgreSQL database.
