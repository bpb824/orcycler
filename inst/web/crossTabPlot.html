Cross tablutation plot

Usage

crossTabPlot(data, xVar, yVar, xlab, ylab, colorPal, xCats = levels(x), yCats = levels(y), px = 1280, py = 560)

Arguments

data
Trip, user, or report table for plotting
xVar
Variable divide into seperate bars
yVar
Variable to fill sections of bars by
xlab
Label for x variable
ylab
Label of y variable
colorPal
Color pallette to use for plotting
xCats
X category names, default is levels of x variable
yCats
Y category names, default is levels of y variable
px
Width of plot in pixels, default is 1280 pixels
py
Height of plot in pixels, default is 560 pixels

Value

None

Description

Cross tablutation plot
